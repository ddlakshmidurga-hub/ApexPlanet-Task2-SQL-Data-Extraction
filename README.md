SQL for Data Extraction project using Python, SQLite, and a cleaned Zomato dataset. This project demonstrates SQL queries such as SELECT, WHERE, ORDER BY, LIMIT, GROUP BY, HAVING, CTEs, and Window Functions for data analysis.
